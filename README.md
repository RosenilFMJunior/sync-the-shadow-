# SYNC of the Shadow // Project Argos

Uma experiência digital mínima sobre o estado de vigilância panóptica, estabilidade operacional e o vazio ativo.

## O Conceito
Inspirado nas dinâmicas de atrito e resolução da física urbana e da psicologia profunda, o **SYNC** monitora a ausência de movimento. Não se trata de uma aplicação de engajamento dinâmico, mas de um manifesto em código sobre a **paralisia deliberada**. 

Ao cessar todo o movimento físico e estabilizar os sensores do dispositivo, o usuário faz a transição do estado de ruído ordinário para a vigília de *Argos*, permitindo que o conteúdo subconsciente emerja na tela.
